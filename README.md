ClimaTrack

O ClimaTrack é um dashboard ambiental desenvolvido para monitoramento de dados climáticos em tempo real.
O projeto é inspirado na ODS 13 da ONU e busca facilitar o acesso a informações sobre desmatamento, temperatura e qualidade do ar.
Funcionalidades:
Monitoramento de indicadores ambientais em tempo real.
Visualização de dados através de gráficos e mapas.
Sistema de login e cadastro de usuários.
Chatbot assistente integrado para suporte.

Tecnologias:
HTML5 e CSS3
JavaScript
Fontes: Poppins e Playfair Display
Fontes de dados: INPE, NASA e IBAMA.
Estrutura de Arquivos:
index.html: Página principal.
login.html: Página de acesso.
login.css e login.js: Estilos e lógica de autenticação.
style.css e script.js: Estilos e lógica globais.
